# Meta Ads Templates

Plantillas operacionales para futuras iteraciones Meta Ads. Mantener archivos sin secretos y copiar a:

```text
ops/meta-ads/<tenant>/<YYYY-MM>-<slug>/
```

Convenciones:

- campanas y anuncios creados por automatizacion empiezan `PAUSED`;
- UTMs deben seguir `docs/ops/utm-convention.md`;
- cada iteracion debe tener `_STATUS.md`;
- toda recomendacion debe separar `propuesta` de `accion ejecutada`.
